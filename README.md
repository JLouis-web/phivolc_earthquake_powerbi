# phivolc_earthquake_powerbi
Philippines Earthquake Monitoring Dashboard - Power BI

This project is an interactive earthquake monitoring dashboard built in Power BI using real-time data from the Pylindol API. It visualizes recent seismic activity across the Philippines, helping users quickly understand earthquake frequency, magnitude distribution, and affected regions.

Features: 

1. Magnitude trends and frequency analysis
2. Interactive map of earthquake locations
3. Time-based filtering (daily, weekly, monthly)
4. Depth and magnitude breakdown

Data Source:

Data is sourced from the Pylindol API, which aggregates earthquake information from PHIVOLCS (Philippine Institute of Volcanology and Seismology).

Tools:

1. Power BI (Dashboard & Visualization)
2. Python (Data extraction via API)
3. Pylindol API (Earthquake data source)

Workflow:

1. Data is fetched from the Pylindol API using Python
2. Data is cleaned and transformed
3. Dataset is loaded into Power BI
